# Chatbot

Super simple chatbot

## How to run

First train the chatbot,

```bash
python3 train.py
```
To start the chatbot, type

```bash
python3 chatbot.py
```

## How to extend chatbot

Open `data.py` and add new intents, questions and possible responses. Then retrain the chatbot again.
